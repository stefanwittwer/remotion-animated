### Spring options

In addition, all options from [Remotion's `SpringConfig`](https://www.remotion.dev/docs/spring#config) can be provided as options.

- [mass: `number`](https://www.remotion.dev/docs/spring#mass)
- [damping: `number`](https://www.remotion.dev/docs/spring#damping)
- [stiffness: `number`](https://www.remotion.dev/docs/spring#stiffness)
- [overshootClamping: `boolean`](https://www.remotion.dev/docs/spring#overshootclamping)

Remotion Animated provides a default smooth spring animation out-of-the-box, if not overwritten by these values.
